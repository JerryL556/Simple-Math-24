# Simple Math 24

Drag-and-drop web version of the 24 game. Use each of the four numbers once, combine with operators/parentheses, and reach 24. Includes an auto-checker and a solver.

## Play locally
Open `index.html` in your browser. No build or install steps required.

## How it works
- Randomly generates a solvable set of four numbers.
- Drag numbers/operators/parentheses into the build area; numbers return to the pool if dragged out.
- Auto-evaluates expressions and highlights success; solver can reveal a solution.
